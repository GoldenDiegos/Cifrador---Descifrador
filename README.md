# CIPHERLAB

Sistema web de cifrado y descifrado de texto.

## Uso

1. Ingresa el conjunto de caracteres a utilizar en el campo **Set activo**
2. Selecciona el modo: **CIFRAR**, **DESCIFRAR** o **RETO**
3. En modo cifrado, elige el método y el módulo deseado
4. Escribe el texto y presiona **EJECUTAR**
5. En modo descifrado, el sistema identifica el método automáticamente
